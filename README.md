# termclr

# Part of ANSI escape sequences

## Set

Bold/bright: \u001b[1m
Dim:         \u001b[2m
Italic:      \u001b[3m
Underline:   \u001b[4m
Blink:       \u001b[5m
Inverse:     \u001b[7m
Conceal:      \u001b[8m
Crossed-out: \u001b[9m
